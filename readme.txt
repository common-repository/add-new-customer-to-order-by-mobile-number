== Description ==

با استفاده از شماره موبایل مشتری، یک مشتری جدید در بخش افزودن سفارش ووکامرس ایجاد نمایید.

== Installation ==

افزونه را از مخزن دانلود کنید و سپس در وردپرس خود آپلود نموده و فعال نمایید

== Use of plugin ==

1. به صفحه افزودن سفارش بروید
2. شماره موبایل مشتری را جستجو نمایید.
3. اگر مشتری وجود نداشت، گزینه ی افزودن مشتری جدید از آدرس صورتحساب را فعال نمایید.
4. به آدرس صورتحساب بروید و تمام اطلاعات مشتری را وارد نمایید.
5. در فیلد تلفن، شماره موبایل مشتری را وارد نمایید.
6. آیتم های سفارش را انتخاب نمایید و سفارش را ذخیره کنید

== Frequently Asked Questions  ==

= آیا این افزونه سرعت سایت من را پایین می آورد؟  =

 تضمین می کنیم که تاثیر افزونه بر روی سرعت سایت شما 'صفر درصد' می باشد.
= آیا می توانم افزونه را از سایت های دیگر دانلود نمایم؟  =

 بله ولی هیچ تضمینی برای اینکه سایت شما هک نشود وجود نخواهد داشت. لذا فقط از طریق مخزن در سایت خود آپلود نمایید.
